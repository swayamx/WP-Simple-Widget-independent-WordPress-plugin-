WP Simple Widget
================

Install:
- Put plugin folder into wp-content/plugins/wp-simple-widget
- Activate plugin via WP admin

Use:
- Admin -> WSW Items to add items
- In any post/page add shortcode: [simple_widget_recent limit=5]
